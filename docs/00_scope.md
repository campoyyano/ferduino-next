# Scope del proyecto ferduino-next

Objetivo:
Evolucionar el proyecto Ferduino original hacia:

- Comunicación local por MQTT
- Integración con Home Assistant
- Portabilidad a otras placas (Mega / Due / Giga)
- Nueva electrónica tipo shield
- Pantalla SPI moderna

Reglas:
- No modificar el código original en firmware/original
- Todo el desarrollo nuevo irá separado
- Documentar antes de cambiar
